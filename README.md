# Sistema de Rutas de Emergencia para Guayaquil

Sistema desarrollado en Python que permite encontrar la ruta más corta hacia servicios de emergencia utilizando el algoritmo de Floyd-Warshall sobre una red vial real obtenida desde OpenStreetMap.

El proyecto está enfocado en la zona de Ceibos y ESPOL, integrando hospitales, estaciones de policía y cuarteles de bomberos para calcular rutas óptimas de atención ante emergencias.

## Características

* Descarga automática de datos desde OpenStreetMap.
* Construcción de una red vial basada en grafos.
* Implementación del algoritmo Floyd-Warshall.
* Búsqueda de hospitales, policías y bomberos cercanos.
* Interfaz gráfica desarrollada con Tkinter.
* Visualización de rutas mediante mapas interactivos con Folium.

## Tecnologías utilizadas

* Python
* Tkinter
* Folium
* OpenStreetMap
* Overpass API
* Floyd-Warshall

## Objetivo del proyecto

Este proyecto fue desarrollado para aplicar conceptos de Matemáticas Discretas y Teoría de Grafos en un problema real de optimización de rutas para servicios de emergencia.

## Cómo ejecutar el proyecto

1. Clonar el repositorio.
2. Instalar las dependencias necesarias.
3. Ejecutar el archivo principal.

```bash
python main.py
```
